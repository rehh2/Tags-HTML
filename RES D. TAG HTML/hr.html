Tag hr – Linha Horizontal

A tag hr é usada em HTML para inserir uma linha horizontal na página. Ela serve como um separador visual entre seções de conteúdo.

A sigla "hr" vem de horizontal rule (regra horizontal). Essa linha pode representar uma mudança de assunto, de seção ou apenas uma divisão estética no layout da página.

Código usado:

A tag hr é uma tag vazia, ou seja, ela não precisa de fechamento. Basta usá-la sozinha no ponto onde deseja que a linha apareça.
