Tag a – Hiperlink

A tag a é usada em HTML para criar hiperlinks, ou seja, textos clicáveis que redirecionam o usuário para outra página, arquivo, seção da mesma página ou até para iniciar um e-mail. Ela é uma das bases da navegação na web.

Para funcionar, a tag a deve usar o atributo href, que define o endereço para onde o link vai levar.

Exemplo:

<a href="https://www.exemplo.com">Visite nosso site</a>

Renderizado: Visite nosso site

Outros usos comuns:

  • Links para arquivos: <a href="arquivo.pdf">Baixar PDF</a>
  • Links internos: <a href="#secao2">Ir para a seção 2</a>
  • E-mail: <a href="mailto:contato@exemplo.com">Enviar e-mail</a>

Você também pode usar o atributo target="_blank" para abrir o link em uma nova aba.
