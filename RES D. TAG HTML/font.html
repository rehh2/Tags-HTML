A Tag <font> em HTML

A tag <font> era usada em versões antigas do HTML para definir formatação visual de texto, como cor, tamanho e tipo de fonte.

Como era usada

Sua sintaxe básica incluía três atributos principais:

<font face="Arial" size="4" color="red">
    Texto formatado
</font>

Atributos

  • color: Definía a cor do texto (nome ou código hexadecimal)
  • face: Especificava a família de fontes
  • size: Determinava o tamanho do texto (1-7)

Status atual

A tag <font> está obsoleta no HTML5. O W3C recomenda o uso de CSS para formatação visual:

<span style="font-family: Arial; font-size: 16px; color: red;">
    Texto formatado
</span>

Alternativas modernas

Use estas propriedades CSS no lugar da tag <font>:

  • font-family (substitui face)
  • font-size (substitui size)
  • color (substitui color)

Por que foi removida?

A tag <font> foi depreciada porque:

  1. Misturava conteúdo com apresentação
  2. Dificultava a manutenção
  3. Não era acessível
  4. CSS oferece muito mais controle e flexibilidade
