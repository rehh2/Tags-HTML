A Tag <blockquote> em HTML

A tag <blockquote> é usada em HTML para marcar citações longas ou blocos de texto que foram retirados de outra fonte.

Como usar

Para usar a tag <blockquote>, envolva o texto citado com a tag de abertura e fechamento:

<blockquote>
    Texto da citação aqui
</blockquote>

Exemplo

Exemplo de citação com blockquote:

"A simplicidade é o último grau de sofisticação."
— Leonardo da Vinci

Atributos

O principal atributo é:

  • cite: Especifica a URL da fonte da citação (opcional). Exemplo: 
    <blockquote cite="https://exemplo.com/fonte">
    Citação aqui
</blockquote>

Diferença para <q>

Enquanto <blockquote> é para citações longas (em bloco), a tag <q> é para citações curtas (inline).

Exemplo com <q>: Isto é uma citação curta
