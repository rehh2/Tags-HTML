Tag Mark – Destaque de Texto

A tag mark é usada para destacar um trecho do texto como se estivesse marcado com um marca-texto amarelo. É útil para chamar atenção para palavras importantes, resultados de busca, ou partes que precisam de ênfase visual.

Exemplo prático:

A prova será aplicada no dia 25 de agosto, às 14h.

O aluno que tirar nota acima de 7 estará aprovado.

Código usado:

<p>A prova será aplicada no dia <mark>25 de agosto</mark>.</p>
<p>Nota mínima: <mark>7</mark></p>

O texto dentro de mark será visualmente destacado com fundo amarelo por padrão. É uma tag de uso semântico e visual, ideal para realces rápidos sem usar CSS.
