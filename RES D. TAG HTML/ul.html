Tag ul em HTML

A tag ul é usada para criar uma lista não ordenada, ou seja, uma lista com marcadores (bolinhas) em vez de números. Ela é composta por itens individuais, definidos com a tag li (list item).

Essa estrutura é ideal para listar informações sem hierarquia numérica, como ingredientes, características ou tópicos.
