DO H1 até o H6

As tags h1 até h6 são usadas para definir títulos e subtítulos em HTML, organizando o conteúdo em níveis de importância. A tag h1 representa o título principal da página, geralmente usado uma única vez para indicar o tema central. Já as tags h2 até h6 são usadas para criar seções e subseções,funcionando como uma hierarquia de conteúdo. Essa estrutura é essencial tanto para a **leitura humana** quanto para os mecanismos de busca (SEO), que usam os títulos para entender a organização da página. Visualmente, os títulos vão diminuindo de tamanho conforme o número aumenta, mas isso pode ser ajustado com CSS. O uso correto dessas tags melhora a acessibilidade, permitindo que leitores de tela naveguem facilmente. Também facilita a leitura rápida, pois destaca os pontos principais do texto. Em blogs, artigos e sites institucionais, elas ajudam a manter o conteúdo limpo e organizado. Cada página deve ter apenas um h1, mas pode conter várias tags dos outros níveis. Em resumo, essas tags não servem apenas para estética, mas para estrutura, navegação e compreensão do conteúdo.
