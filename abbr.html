A Tag <abbr> em HTML

A tag <abbr> é usada para definir abreviações e siglas em documentos HTML.

Como usar

Use a tag <abbr> envolvendo a abreviação e incluindo o atributo title com a explicação completa:

<abbr title="HyperText Markup Language">HTML</abbr>

Atributo principal

O atributo title é obrigatório e deve conter o significado completo da abreviação:

  • <abbr title="World Wide Web">WWW</abbr>
  • <abbr title="Cascading Style Sheets">CSS</abbr>

O HTML é a linguagem base da web.

Importância para acessibilidade

A tag <abbr> é importante porque:

  1. Fornece contexto para leitores de tela
  2. Mostra o significado completo quando o usuário passa o mouse
  3. Melhora a compreensão do conteúdo
  4. Ajuda em mecanismos de busca

Diferença para <acronym>

Antigamente existia a tag <acronym> para siglas, mas ela foi removida no HTML5. Agora usamos apenas <abbr> para:

  • Abreviações (ex: "Dr." para "Doutor")
  • Siglas (ex: "ONU" para "Organização das Nações Unidas")
  • Iniciais (ex: "FBI" para "Federal Bureau of Investigation")

Dica de estilo

Por padrão, alguns navegadores mostram abreviações com:

  • Linha pontilhada inferior
  • Cursor em forma de interrogação ao passar o mouse

Mas você pode personalizar isso com CSS se desejar.
